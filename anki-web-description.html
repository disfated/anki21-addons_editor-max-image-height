This add-on allows limiting the height of the images displayed in the Anki card editor so that all the fields are (possibly) visible together. To change the maximum image height, go to Anki 2.1: Tools > Add-ons > Maximum image height in card editor > Config and set the variable max-height equal to the pixels you want. Anki 2: Tools > Add-ons > max_img_height > Edit.. and set the variable MAXHEIGHTPX equal to the pixels you want Changelog Anki 2.1 v1.0.1 (24/02/2019) Fix image HTML modified by addon v1.0 (09/07/2018) Initial release Anki 2 v0.1 (12/04/2015) Initial release Issues Report any bug in the GitHub issues page of the project.
