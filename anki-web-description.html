This add-on allows limiting the height or width of the images displayed in the Anki card editor so that all the fields are (possibly) visible together. To change the maximum image height or width, go to Anki 2.1: Tools > Add-ons > Maximum image height and width in card editor >. Each property is documented in the configuration dialog.
